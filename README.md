## 🧠 About Me

> *"Security is not a product, but a process."* — I live by this.

I'm **Yuvaraj S**, a fresher passionate about **Cyber Security**, ethical hacking, and building secure systems from the ground up. I love exploring vulnerabilities, understanding attack surfaces, and crafting tools that make the digital world safer.

Beyond the terminal, I share my knowledge through **content creation** — breaking down complex security concepts into digestible content on **YouTube**, **Medium**, and **Instagram**.

- 🔐 **Currently learning:** Penetration Testing, Network Security & CTF challenges
- 🛠️ **Building:** Security tools and scripts in Python & Bash
- 📝 **Writing about:** Cybersecurity, ethical hacking & tech on Medium
- 🎯 **Goal:** Land a role in Red Teaming / SOC / Threat Intelligence
- ⚡ **Fun fact:** I can spend 6 hours debugging a script and call it "fun"
